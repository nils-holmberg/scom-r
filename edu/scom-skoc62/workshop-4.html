4.1 Multiple regression analysis

  1. Create a model using multiple regression analysis Analyze -> Regression -> Linear.

  2. Under Dependent, enter your dependent variable (informationaccessindex, Workshop #3 task 1a). Under Independent(s), enter your independent variables: Age in years (see Workshop #2 task 2), dum_female, Org_type, dum_manager and dum_communicator (Workshop #3 task 3). Click OK.

  3. Choose Method: Enter.
